la prima cosa da fare è suddividere in blocchi logici la pagina,identificando le parti comuni da applicare a tutte le pagine

    le varie aree saranno quindi suddivise in:
  • contenuti di intestazione del tag
  • contenuti centrali variabili nel tag (corpo principale)
  • contenuti di fine pagina comuni <--!footer!-->

formattazzione del codice

    il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con i CSS. perciò i passi saranno:
  1. Proggettare il markup assegnando ad ogni specifico testo il tag appropriato
  2. Scrivere HTML dai tag estrerni a quelli interni e dall'alto verso il basso
  3. Applicare gli stili CSS

L'applicazione di stili CSS è creciuta negli anni fino a raggiungere la diffusione pressoché totale.
